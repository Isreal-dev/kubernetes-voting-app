# Kubernetes Voting App

A production-style microservices voting application built with Docker and Kubernetes.

## Architecture

- Vote Service (Flask)
- Result Service
- Worker Service
- Redis (message queue)
- PostgreSQL (database)

## Tech Stack

- Python (Flask)
- Docker
- Kubernetes
- Redis
- PostgreSQL

## Run Locally (Docker Compose)

```bash
docker-compose up --build
